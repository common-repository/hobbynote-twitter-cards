Aider les utilisateurs à implémenter et personnaliser les Twitter Cards facilement.

== Description ==

Ce plugin permet d'implémenter et de personnaliser facilement les Twitter Cards sur votre blog.

Le module va ajouter directement les balises meta nécessaires à l'implémentaton des Twitter Cards. 
Vous pourrez définir dans le backoffice du module le compte Twitter associé au site web, le type de Twitter Card souhaité (summary, summary_large_image, photo), les champs titre et description (pouvant être récupérés directement à partir des données entrées par le plugin WPSEO by Yoast ou ALL In One SEO), une image pour votre Homepage et vos posts.
 
Vous trouverez également 3 liens dans le menu du plugin :
- Documentation : Lien redirigeant vers la documentation des Twitter Cards
- Validation : Lien pour valider vos Twitter Cards (il faut valider obligatoirement la homepage pour que votre nom de domaine soit validé)
- Debuger : Lien redirigeant vers un outil développé par notre agence. Il s'agit d'un outil nommé "SMO Analyzer" qui permet d'analyser la présence des Twitter Cards sur votre site. 

== Installation ==

Pour installer le plugin, il faudra vous rendre dans le backoffice de votre blog, vous rendre dans la catégorie "Extensions", cliquer sur Ajouter et uploader le module.

== Frequently Asked Questions ==

= Comment configurer le plugin ? =

Il faut se rendre dans le menu Twitter Cards dans votre backoffice sur la gauche et compléter les différentes sections.
